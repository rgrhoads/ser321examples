You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html e.g. http://54.215.239.18/file/sample.html

  • /json -- returns a json of the /random request
      e.g. http://54.215.239.18/json

  • /random -- returns index.html
      e.g. http://54.215.239.18/random

  • /multiply -- returns product of num1 and num2
      e.g. http://54.215.239.18/multiply?num1=2&num2=4

  • /github -- returns JSON array of github repositories
      e.g. http://54.215.239.18/github?query=users/rgrhoads/repos

  • /free-games -- returns JSON array of free games and content
      e.g. http://54.215.239.18/free-games?platform=steam&type=game

      accepted platforms:
        pc, steam, epic-games-store, ubisoft, gog, itchio, ps4, ps5, xbox-one, xbox-series-xs, switch, android, ios, vr, battlenet, origin, drm-free, xbox-360

      accepted types:
        game, loot, beta

  • /remote-jobs -- returns JSON array of open remote jobs
      e.g. http://54.215.239.18/remote-jobs?count=5&geo=usa&industry=engineering

      accepted geos:
        apac, emea, latam, argentina, australia, austria, belgium, brazil, bulgaria, canada, china, costa-rica, croatia, cyprus, czechia, denmark, estonia, europe
        finland, france, germany, greece, hungary, ireland, israel, italy, japan, latvia, lithuania, mexico, netherlands, new-zealand, norway, philippines, poland
        portugal, romania, singapore, slovakia, slovenia, south-korea, spain, sweden, switzerland, thailand, turkiye, uk, united-arab-emirates, usa, vietnam

      accepted industries:
        business, copywriting, supporting, data-science, design-multimedia, admin, accounting-finance
        hr, marketing, management, dev, seller, seo, smm, engineering, technical-support, web-app-design
